Finite State Machines

Description

Finite State Machines is a workshop project made to help gain a better understanding into implementing finite state machines in GameMaker programming language (GML).

NOTE: A majority of this game was provided by the assignment for the ITSC-5230 class. My implementation focused on adding the player controller's state management.

Instructions

Controls:
Movement: Use the arrow keys to move the character.
Pushing: When near a block, press 'E' to start or stop pushing the block.
Dancing: Press 'Space' to make the character dance. While dancing, the character cannot perform other actions.
